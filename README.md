# M-CITY

M-CITY のトップページを再現した静的サイトです。

## 内容

- M-CITY へようこそ
- M-CITY 簡単アクセスガイド
- M-CITY ってなに？

プロフィールと Slack 紹介セクションは含めていません。

## ローカルで確認

`index.html` をブラウザで開くか、簡易サーバーを立てて確認できます。

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

そのあと `http://127.0.0.1:4173` を開いてください。

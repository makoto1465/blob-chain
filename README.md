# Gather 1.0 Participant Guide

Gather 1.0 Classic に招待された参加者向けの日本語導入ページです。

PC版とスマホ版を分けて、招待リンクから入室する流れ、カメラ・マイク許可、名前入力、アバター設定、基本操作、トラブル時の確認点をまとめています。

## Local preview

This is a static site. Open `index.html` directly, or run a small local server:

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

Then visit `http://127.0.0.1:4173`.
